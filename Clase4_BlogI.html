Nota Informativa sobre HTML

¿Qué es HTML?

HTML, que significa HyperText Markup Language, es el estándar para la creación de páginas web. Fue desarrollado por primera vez por Tim Berners-Lee en 1991.

Este video proporciona una visión general de las nuevas características de HTML5.

¿Por qué es importante HTML?

HTML es fundamental porque estructura y da formato a los contenidos de la web. Utiliza etiquetas para definir elementos como encabezados, párrafos, listas, enlaces, imágenes y mucho más.

Escucha nuestro podcast sobre HTML.

Etiquetas comunes en HTML

Etiqueta Descripción
p Define un párrafo de texto.
h1-h6 Define encabezados de diferentes niveles.
a Define un enlace o hipervínculo.
ul y li Define listas desordenadas y sus elementos.
Más información sobre HTML5

HTML5 incluye nuevas características, mejoras y un mejor soporte para multimedia como audio y vídeo. También introduce nuevos elementos semánticos que mejoran la estructura del documento.

  • Etiquetas semánticas como <article>, <section>, <header>, y <footer>.
  • Soporte nativo para audio y vídeo con <audio> y <video>.
  • Mejora del manejo de formularios con nuevos tipos de entrada.
